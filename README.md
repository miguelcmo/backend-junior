# Desarrollo Backend Junior

## Programa Stud-IA - Alcaldia de Medellin

**Duracion:** 236 horas | **Modalidad:** 60% Virtual - 40% Presencial
**Nivel:** Basico-Intermedio

---

## Objetivo del Programa

Capacitar en los fundamentos del desarrollo backend, disenar y construir APIs y servidores, integrar bases de datos, aplicar metodologias agiles y DevOps, y desarrollar habilidades para la resolucion de problemas tecnicos.

---

## Stack Tecnologico

| Area | Tecnologias |
|------|-------------|
| **Frontend** | HTML5, CSS3, JavaScript (ES6+), TypeScript, React.js, Next.js, Tailwind CSS |
| **Backend** | Node.js, Express.js, Next.js API Routes, RESTful APIs |
| **Base de Datos** | PostgreSQL, Prisma ORM, SQL |
| **Autenticacion** | NextAuth.js, JWT, bcrypt |
| **DevOps** | Git, GitHub, Docker, GitHub Actions, Vercel, Railway |
| **Testing** | Jest, Vitest, React Testing Library |

---

## Modulos del Programa

### Fase 1: Fundamentos Web (48h)
- Fundamentos de Desarrollo Web
- HTML5 y Estructura Web
- CSS3 y Diseno Responsive
- JavaScript Moderno (ES6+)

### Fase 2: TypeScript y React (30h)
- TypeScript Fundamentals
- Introduccion a React.js

### Fase 3: Backend con Node.js (32h)
- Introduccion a Node.js
- Express.js y APIs RESTful

### Fase 4: Bases de Datos (32h)
- PostgreSQL
- ORM con Prisma

### Fase 5: Next.js Full-Stack (54h)
- Next.js Framework
- Autenticacion y Seguridad
- Integracion Full-Stack

### Fase 6: Automatizacion y DevOps (16h)
- Testing
- CI/CD
- Deployment

### Fase 7: Proyecto Final (25h)
- Aplicacion Full-Stack completa
